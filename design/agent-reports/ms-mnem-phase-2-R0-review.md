- **[M1] `crates/ms-cli/src/cmd/inspect.rs:97-111` — rule-10 entr arm is now `kind`-gated; for `InspectKind::Unknown` the payload-length check no longer runs.** Pre-Phase-2 the guard was `if tag_bytes == TAG_ENTR && !VALID_ENTR_LENGTHS.contains(...)`; now it is `InspectKind::Entr if tag_bytes == TAG_ENTR { ... }` with `_ => {}`.
